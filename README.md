# Диплом: LaTeX-проект ВКР

Этот репозиторий содержит исходники выпускной квалификационной работы, оформляемой в LaTeX.  
Проект адаптирован под текущие требования вуза и ориентирован на удобную модульную работу: каждая крупная часть текста хранится в отдельном файле и подключается в основной документ через `\input`.

## Общая логика проекта

Основной файл сборки — `00_main.tex`.  
Именно он определяет структуру документа, порядок подключения глав, библиографии и приложений.

Класс документа — `99_itmo-student-thesis.cls` или его локальная модификация.
Файл класса в данном проекте основан на исходном шаблоне [itmo-student-thesis](https://github.com/mbuzdalov/latex-styles/tree/master/itmo-student-thesis) и был адаптирован под текущие требования оформления ВКР 2026 года.

Автору исходного шаблона выражается благодарность за проделанную работу и создание удобной основы, которая существенно упростила доработку класса под конкретные задачи данного проекта!

В классе сосредоточены:
- подключение пакетов;
- настройка шрифтов, полей и интервалов;
- оформление заголовков, оглавления, приложений;
- настройки библиографии;
- настройки листингов кода.

Текст ВКР разбит по смысловым файлам, чтобы было удобно редактировать главы независимо друг от друга.
